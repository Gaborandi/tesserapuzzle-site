The Core Idea

A region is correct only when its shape and rule both hold.

Every mode asks the same elegant question: which cells belong together? The answer changes when the rule becomes numbers, color, or exact equations.

Logic

Reverse Sudoku

Draw equal-size connected regions where no number repeats. Clean, readable, and immediately understandable.

1234 3412 4231 4213
Chroma

Color Mixing

Group primary colors into regions that mix into a target shade. It feels tactile without losing puzzle rigor.

RYBO GRPB YGRP BOYG
Equate

Spatial Math

Make regions whose numbers combine to a target value. Addition, multiplication, subtraction, and exact division all matter.
